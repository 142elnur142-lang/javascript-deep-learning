Classes - Классы ES6

Синтаксис классов и создание объектов

Базовый синтаксис

class Animal {
    constructor(name) {
        this.name = name;
    }
    
    speak() {
        console.log(`${this.name} makes a sound`);
    }
}

const dog = new Animal('Buddy');
dog.speak();

Наследование

class Dog extends Animal {
    bark() {
        console.log(`${this.name} barks`);
    }
}

const dog = new Dog('Rex');
dog.speak(); // Наследованный метод
dog.bark();  // Свой метод
