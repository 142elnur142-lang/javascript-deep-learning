Callbacks - Обратные вызовы

Обратные вызовы и callback hell

Что такое Callback?

Callback это функция передаваемая как аргумент для асинхронного выполнения.

function fetchData(callback) {
    setTimeout(() => callback('data'), 1000);
}

fetchData((result) => console.log(result));
