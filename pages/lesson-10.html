Destructuring - Деструктуризация

Деструктуризация массивов и объектов

Деструктуризация массивов

const [a, b, c] = [1, 2, 3];
console.log(a); // 1
console.log(b); // 2

Деструктуризация объектов

const { name, age } = { name: 'Alice', age: 30 };
console.log(name); // 'Alice'
console.log(age);  // 30
